# TMDB Commercial License Requirements

## ⚠️ Important Notice

**This project uses TMDB API for portfolio/demo purposes only.**

**For production deployment with commercial use, a commercial license from TMDB is REQUIRED.**

---

## 📋 License Requirements

### Portfolio/Demo Use (Current Project)

✅ **Allowed:**
- Non-commercial use
- Portfolio projects
- Demo applications
- Educational purposes

**Requirements:**
- Attribution (logo + text)
- Link to https://www.themoviedb.org
- Cache limit: maximum 6 months

### Production Use (Commercial)

❌ **Requires Commercial License:**
- Applications with ads (even free apps)
- Paid/subscription applications
- Services generating revenue (even indirectly)
- "Destination websites" (search engines, chatbots)
- AI/ML model training using TMDB data

**Contact:** sales@themoviedb.org

## 📝 Attribution Requirements

Even with a commercial license, you must:

### 1. Display TMDB Logo

- Use only approved versions of the logo
- Logo must be less prominent than your own logo
- Do not modify (colors, rotation, etc.)

### 2. Attribution Text

> "This [website, program, service, application, product] uses TMDB and the TMDB APIs but is not endorsed, certified, or otherwise approved by TMDB."

### 3. Link

Include a link to: https://www.themoviedb.org
